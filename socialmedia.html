Social Media - Projeto de Campanha

Este projeto foi desenvolvido para uma campanha de social media voltada para o lançamento de um novo produto. A ideia central foi criar uma identidade visual que chamasse a atenção e fosse facilmente compartilhada nas redes sociais.

O processo de criação começou com uma pesquisa de público-alvo, seguida de brainstorms sobre cores, fontes e elementos gráficos que se alinhavam com a identidade da marca. A partir dessas discussões, criamos uma série de postagens que mantinham a consistência visual, usando um estilo vibrante e moderno.

Processo Criativo

Durante a criação deste projeto, enfrentamos o desafio de manter a clareza das informações, mesmo com o uso de uma paleta de cores ousada. A equipe optou por focar em uma abordagem minimalista para garantir que a mensagem fosse o ponto principal das postagens.
